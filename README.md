# COMP 474 — Academic Course Advisor

## How to Run

1. Open **CLIPS IDE**

2. In the CLIPS dialog, load the files (run this with the path where you've cloned these files):
```
(load "/path/to/deliverable-1/factbase.clp")
(load "/path/to/deliverable-1/rulebase.clp")
```

3. Run the system (run both of these commands):
```
(reset)
(run)
```

## To Restart
```
(reset)
(run)
```

## To Clear Everything
```
(clear)
```
Then reload the files and run again.
